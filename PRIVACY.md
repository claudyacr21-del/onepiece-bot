# One Piece Bot Privacy Policy

Last updated: May 21, 2026

This Privacy Policy explains what information One Piece Bot collects and how it is used.

## 1. Information Collected

One Piece Bot may collect and store the following information:

- Discord user ID
- Discord username
- Server ID
- Game progress
- Cards, items, weapons, fruits, fragments, and currency
- Cooldowns, quests, arena data, raid data, and command-related game data

The bot does not collect passwords, email addresses, payment card information, or private Discord messages.

## 2. How Information Is Used

The collected data is used only to operate the bot's game systems, including:

- Saving player progress
- Running commands
- Managing cooldowns
- Handling inventory, cards, trades, raids, arena, and rewards
- Preventing abuse, bugs, or exploits
- Providing support and troubleshooting

## 3. Data Sharing

One Piece Bot does not sell user data.

Game data may be visible inside Discord commands when users interact with the bot, such as profiles, leaderboards, trades, raids, or public game results.

Data may be accessed by the bot owner or admin team when needed for support, moderation, bug fixing, or abuse prevention.

## 4. Data Storage

Data is stored for as long as needed to provide the bot's features.

If the bot is removed from a server or a user stops using the bot, some data may remain unless removal is requested.

## 5. Data Removal

Users may request removal of their bot data by contacting the bot owner or opening a support ticket in the official Discord server.

## 6. Security

Reasonable steps are taken to protect bot data, but no system is guaranteed to be completely secure.

## 7. Changes

This Privacy Policy may be updated at any time. Continued use of the bot means you accept the updated policy.

## 8. Contact

For questions or data removal requests, contact the bot owner or open a support ticket in the official Discord server.
